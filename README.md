## Prerequisites
Before you begin, ensure you have the following installed:
- Java Development Kit (JDK) 8 or higher
- Maven 3.6+ or Gradle 6.0+ (depending on build tool used)
- Git for version control

## Installation

### Clone the Repository
```bash
git clone https://github.com/exelynt-learning-platform/second-round-assignm-final-14262-ganesh.git
cd second-round-assignm-final-14262-ganesh
```

### Build the Project

#### Using Maven
```bash
mvn clean install
```

#### Using Gradle
```bash
gradle build
```
